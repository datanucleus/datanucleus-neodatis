DataNucleus "NeoDatis" provides persistence of Java objects to NeoDatis datastores. It builds on top of the DataNucleus "core" jar supporting specific features of NeoDatis datastores. The code can be utilised with either JDO or JPA persistence APIs.

Support for this persistence layer is provided by way of a Free Forum (no guarantee of availability) or via Commercial Support from DataNucleus. Please refer to the NeoDatis Website for details of NeoDatis datastores.
